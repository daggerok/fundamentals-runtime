# Документация / Documentation

Этот каталог содержит документы, подготовленные в рамках архитектурного ревью приложения **fundamentals**.

## Доступные документы

| Файл | Язык | Назначение |
|------|------|------------|
| [findings.en.md](findings.en.md) | English | Архитектурный обзор (технологии, библиотеки, проблемы, рекомендации) |
| [findings.ru.md](findings.ru.md) | Русский | Архитектурный обзор |
| [REQUIREMENTS.en.md](REQUIREMENTS.en.md) | English | Структурированные требования к приложению (для работы с AI-агентами) |
| [REQUIREMENTS.ru.md](REQUIREMENTS.ru.md) | Русский | Требования к приложению |

## Как использовать

- **findings.** — для понимания текущего состояния приложения, выявленных проблем и рекомендаций.
- **REQUIREMENTS.** — для общения с AI-агентами при дальнейшей разработке, форке showcase-версии или добавлении новых фич.

> Эти документы были созданы автоматически агентом в рамках ревью и могут быть обновлены по мере развития проекта.
